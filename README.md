# PL_SP_Lock_SVN
It is a subversion dump file.  Do not edit this file.
To create a subversion repo from this do the following.
Install subversion, including CLI if using TortoiseSVN.
Clone into a directory - C:\SP_Lock for example.
In a terminal execute the following commands:
svnadmin create SP_Lock
svnadmin load SP_Lock < SP_Lock_svn.dump
When the text finishes loading open the repo SP_Lock.
